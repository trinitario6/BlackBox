Google Drive Photos
Shared household photo backup

Photos upload to a shared Google Drive folder. All household members sign in with their own Google account and access the same folder.

Setup guide

1. Go to console.cloud.google.com → New project

2. Enable Google Drive API

3. Credentials → Create → OAuth 2.0 Client ID → Web application

4. Add your GitHub Pages URL to Authorised JavaScript origins

5. Create a shared folder in Google Drive → share it with all household members → copy the folder ID from its URL

6. Paste Client ID and Folder ID above — each person does this once on their device
