Not connected

Box data syncs to a shared Google Sheet — one central place all household members read and write. Photos upload to a shared Google Drive folder. Everyone uses the same Sheet ID and folder; each person signs in with their own Google account.

Create a Web app OAuth 2.0 credential in Google Cloud Console with your GitHub Pages URL as an authorised origin.

Create a Google Sheet, share it (Editor access) with all household members, and paste the ID from the URL: docs.google.com/spreadsheets/d/SHEET_ID/edit

Create a Drive folder, share it (Editor) with all members, open it and copy the ID from the URL: drive.google.com/drive/folders/FOLDER_ID

Step-by-step setup (5 min)
1Go to console.cloud.google.com → create project BlackBox
2Enable Google Sheets API and Google Drive API in APIs & Services → Library
3OAuth consent screen → External → add scopes spreadsheets and drive.file
4Credentials → Create OAuth Client ID → Web app → add your GitHub Pages URL as authorised origin
5Create a Google Sheet. Paste the Sheet ID above. Share it as Editor with all household members.
6Create a Drive folder for photos. Share it as Editor. Paste the folder ID above.
7Each household member opens Settings on their device, enters the same 3 values, taps Save & Sign In.
