Sync — Google Sheets
Not connected

Box data syncs to a shared Google Sheet. Photos go to a shared Drive folder. Everyone uses the same IDs; each person signs in with their own Google account.

Create a Web app OAuth 2.0 credential in Google Cloud Console with your GitHub Pages URL as an authorised origin.

Create & share a Sheet (Editor). ID is in the URL: spreadsheets/d/SHEET_ID/edit

Create & share a Drive folder (Editor). ID is in URL: drive/folders/FOLDER_ID

Step-by-step setup
1Go to console.cloud.google.com → create project BlackBox
2Enable Google Sheets API and Google Drive API
3OAuth consent screen → External → add scopes spreadsheets and drive.file
4Credentials → OAuth Client ID → Web app → add your GitHub Pages URL as authorised origin
5Create & share a Google Sheet (Editor access) → paste Sheet ID above
6Create & share a Drive folder (Editor) → paste Folder ID above
7Each household member: open Settings, enter same 3 values, tap Save & Sign In
