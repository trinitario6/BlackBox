Black Box uses Firebase to sync boxes and photos across all household members in real time. No sign-in needed — just paste the same config on each device.

Paste the full firebaseConfig object from your Firebase project settings.

How to set up Firebase (free, ~5 min)
1Go to console.firebase.google.com → Add project (use your Google account)
2In the left menu, click Firestore Database → Create database → choose Start in test mode → pick a region → Enable
3In the left menu, click Storage → Get started → Start in test mode → Done
4Click the gear icon → Project settings → scroll to Your apps → click </> (Web) → register with any nickname → copy the firebaseConfig block
5Paste the config JSON above (just the object inside the curly braces) and tap Save & Connect
6Share the same JSON with everyone in the household. They paste it in their Settings too. That's it — no accounts, no passwords.
